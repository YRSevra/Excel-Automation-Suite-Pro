# 🚀 Excel Automation Suite Pro

> **Automatically Clean • Analyze • Visualize • Generate Reports from Excel Files**

A professional Python Automation project built using **Python, Pandas, Streamlit and Plotly** that automatically cleans Excel/CSV datasets, performs advanced analysis, generates business insights, creates reports, and provides an interactive dashboard.

---

## ✨ Features

- 📂 Excel (.xlsx) & CSV Support
- 🧹 Automatic Data Cleaning
- 🤖 AI Dataset Detection
- 📊 Interactive Dashboard
- 📈 Advanced Charts
- 📉 Correlation Analysis
- 📋 Statistical Summary
- 📦 Executive Business Dashboard
- 🧠 Smart AI Recommendations
- 🚨 Outlier Detection
- 🗂 Upload History
- 📄 Automatic PDF Report Generation
- 📥 Excel Export
- 💾 SQLite Database Support

---
